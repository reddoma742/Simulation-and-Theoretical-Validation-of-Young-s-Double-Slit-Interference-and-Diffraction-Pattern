# Simulation-and-Theoretical-Validation-of-Young-s-Double-Slit-Interference-and-Diffraction-Pattern
"Modeling Light Wave Interference: A Comprehensive Computational Approach

## Overview
This project provides a precise numerical simulation of Young's double-slit experiment. It integrates both **interference** and **diffraction** (Fresnel/Fraunhofer envelope) effects to create a physically accurate representation of light wave behavior.

## Scientific Validation
The numerical results are validated against the theoretical interference condition `y = m * λ * L / d`. As shown in the simulation results, the theoretical peak positions (indicated by dashed lines) show a perfect match with the numerical intensity maxima.

## Features
- **Physical Model**: Implements wave superposition combined with a diffraction envelope using `sinc` functions.
- **Scientific Validation**: Validates numerical peaks against classical optical theory.
- **Visualization**: Generates high-quality plots of the intensity distribution.
